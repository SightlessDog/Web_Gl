*Anleitung*
im Rest Ordner sind die Quelledateien Dateien zu finden (die JavaScript Datein) . das ganze wird mit webpack(webpack.config.js) 
kompiliert und in nur eine Datei reingepackt(bundle.js) die wird nachher geladen, um die App zu starten.

I-Node Variante
Um die App zu starten müssen erstmal die Folgenden Commands in bash eingeben: 
1 . npm install 
2 . http-server
3 . im Browser können Sie jetzt eine von den angezeigten Adressen eingeben, um die App zu starten 


II-Variante: mit python
python 3 : python -m http-server PORT(z.B 8080)
